# Store listing draft

## Title

Pi Coding Agent

## Short description

Raycast companion for Pi: quick ask, session search, project launch, transforms, and terminal handoff.

## Long description

Pi Coding Agent brings the Pi terminal coding harness into Raycast. Ask quick questions, browse local Pi sessions, launch recent projects, transform selected text, inspect prompt/skill resources, and start Git-aware workflows without losing the observability of a real terminal.

Risky workflows hand off to visible Pi terminal sessions. Quick Ask and Transform Selection can run in either Fast No Tools mode or Safe Read-Only Project Tools mode.
