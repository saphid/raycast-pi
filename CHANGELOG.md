# Changelog

## 0.1.0 - 2026-04-29

Initial public-ready build.

- Ask Pi with visible streamed responses.
- Copy-only Ask Pi variant.
- Project launcher from Pi session history.
- Session browser with preview, resume, and fork actions.
- Selected-text transforms.
- Git workflow terminal handoffs.
- Pi resource browser for prompts and skills.
- Menu bar status item.
- Cached, bounded session indexing to avoid loading large JSONL files into Raycast workers.
- Preferences for Ask execution mode and max indexed sessions.
